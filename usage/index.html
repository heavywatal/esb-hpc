Metal — HPC cluster of ESB, SOKENDAI

Usage

Getting Started

  1. Complete the online registration form.
  2. Accept the invitation to the user mailing list.
  3. Login to the server with the preset password: ssh <USERNAME>@metal.campus.soken.ac.jp
  4. Change your password: passwd
  5. Configure SSH public key

Basics

  • All the computational tasks should be managed by the PBS job scheduler (see below). Do NOT execute programs directly on the head node.
  • Your home directory ~/ is shared with all the nodes. You don’t need to transfer your data to each compute node.
  • 100GB disk space is allocated for each user. The size may be changed in the future.
